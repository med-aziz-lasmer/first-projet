# Gestion Étudiant Tunisien

Mini projet Java pour calculer la moyenne d’un étudiant tunisien avec coefficients.

## 📌 Fonctionnalités
- Saisie du nom et prénom
- Saisie des modules
- Saisie des notes et coefficients
- Calcul de la moyenne générale

## 🧮 Méthode de calcul
La moyenne est calculée comme suit :

moyenne = somme(note × coefficient) / somme(coefficients)

## 🛠️ Technologies utilisées
- Java
- IntelliJ IDEA
